# SimpleGeometry-Demo

## Description
Demo project showing usage of the [SimpleGeometry](https://github.com/donavoncade/SimpleGeometry)
package in SwiftUI.

SimpleGeometry makes the geometric properties of any view easily accessible throughout
your project.
